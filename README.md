## Local workflow

### Check status

```bash
git status
```

### Stage changes

```bash
git add .
```

### Commit

```bash
git commit -m "Describe your changes"
```

Example:

```bash
git commit -m "Refine landing page"
```

### Publish

```bash
git push
```

---

## If push is rejected

```bash
git pull --rebase
git push
```

## Expected final status

```bash
git status
```

